UnstructuredGrid Topology Example

This example demonstrate the creation of UnstructuredGrid file (.vtu).

The mesh is composed by 11 elements of different types with a total of 27 nodes coordinates.

Let us assume that all the necessary UnstructuredGrid variables have been defined and initialized, the following is a pseudo code showing how to create XML file:

  • file initialization:
    E_IO = VTK_INI_XML(output_format = 'Binary', filename = 'XML_UNST.vtu', mesh_topology = 'UnstructuredGrid')
  • mesh saving of the first (unique) piece:
    E_IO = VTK_GEO_XML(NN = Nn, NC = Ne, X = x, Y = y, Z = z)
  • mesh connectivity saving of the first (unique) piece:
    E_IO = VTK_CON_XML(NC = Ne, connect = connect, offset = offset, cell_type = cell_type )
  • initializing the file tag data-container for saving data associated to the current piece:
    E_IO = VTK_DAT_XML(var_location = 'node', var_block_action = 'open')
  • data saving:
    E_IO = VTK_VAR_XML(NC_NN = Nn, varname = 'scalars', var = v)
    E_IO = VTK_VAR_XML(NC_NN = Nn, varname = 'vector', varX=v_X,varY=v_Y,varZ=v_Z)
  • closing the file tag data-container:
    E_IO = VTK_DAT_XML(var_location = 'node', var_block_action = 'close')
  • closing current piece:
    E_IO = VTK_GEO_XML()
  • file finalization:
    E_IO = VTK_END_XML()

This example is contained into Test_Driver.f90, in particular the subroutine test_unst contains all the data and code to run this example.
